## Veröffentlichungs-Gate

Vor Veröffentlichung auf der produktiven Website muss ein Verantwortlicher prüfen:

1. ob der temporäre Demo-Link durch eine dauerhafte Domain ersetzt wurde;
2. ob keine Testdaten, Zugangsdaten oder privaten Informationen sichtbar sind;
3. ob Impressum, Datenschutz und Cookie-Hinweise der Website unverändert korrekt bleiben;
4. ob die verlinkten Dokumente tatsächlich der gewünschten Öffentlichkeit entsprechen.
